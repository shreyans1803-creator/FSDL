# 🚀 Freelancer Marketplace System

## 📌 Overview
This is a full-stack web application where clients can post projects and freelancers can view them.

---

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

---

## ⚙️ Features
- Post new projects
- View all available projects
- Data stored in MongoDB
- Real-time project listing

---

## 📁 Project Structure
FSDL/
├── backend/
├── frontend/

---

## ▶️ How to Run

### Backend
cd backend  
node server.js  

### Frontend
cd frontend  
npm start  

---

## 🌐 URLs
- Frontend: http://localhost:3000  
- Backend: http://localhost:5000
